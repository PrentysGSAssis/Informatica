Guia de Configuração de Favoritos

Organizando seu Ambiente de Trabalho Digital no Google Chrome

1. Exibindo a Barra de Favoritos

O primeiro passo é garantir que a sua Barra de Favoritos esteja sempre visível. Ela aparecerá logo abaixo da barra de endereço.

  1. No Google Chrome, clique no ícone de três pontos verticais (⋮) no canto superior direito para abrir o menu.
  2. Posicione o mouse sobre "Favoritos e listas".
  3. No menu que aparecer, clique em "Exibir barra de favoritos".
Dica de Atalho: Você pode exibir ou ocultar a Barra de Favoritos a qualquer momento usando o atalho de teclado: Ctrl + Shift + B.

2. Adicionando Favoritos Principais

Agora, vamos adicionar alguns sites úteis diretamente na Barra de Favoritos para acesso rápido.

Para cada site da lista abaixo, siga estes passos:

  1. Encontre o site: Use a busca do Google para encontrar o site oficial.
  2. Acesse o site: Clique no link correto para abrir a página principal do site.
  3. Favorite a página: Com o site aberto, clique no ícone de estrela (☆) no lado direito da barra de endereço.
  4. Salve na Barra de Favoritos: Na pequena janela que aparecer, certifique-se de que a "Pasta" selecionada seja a "Barra de favoritos". Clique em "Concluído".

Sites para adicionar:

  • YouTube
  • Wikipedia
  • Google Sala de Aula
  • Ninite
  • ChatGPT
  • Gemini
  • Copilot
  • Lightshot

3. Organizando com Pastas (Nível 1)

Para manter os favoritos organizados, vamos criar pastas por assunto. A primeira será para ferramentas de desenvolvimento web.

3.1 - Criar a pasta "DESENVOLVIMENTO WEB"

  1. Pesquise e acesse o site oficial do Visual Studio Code.
  2. Clique na estrela (☆) para favoritar.
  3. Na janela de favoritos, clique no menu suspenso ao lado de "Pasta" e selecione "Outra pasta...".
  4. Na nova janela, clique no botão "Nova pasta" no canto inferior esquerdo.
  5. Digite o nome DESENVOLVIMENTO WEB e clique em "Salvar". O site do Visual Studio Code será salvo dentro desta nova pasta.

3.2 - Adicionar mais sites à pasta

Agora, adicione os seguintes sites à pasta que você acabou de criar:

  • GIMP
  • EMOJIPEDIA
Como fazer: Para cada site, clique na estrela (☆) para favoritar. Na janela que aparecer, clique no menu "Pasta" e simplesmente selecione a pasta "DESENVOLVIMENTO WEB" que já existe na lista. Clique em "Concluído".

4. Organizando com Sub-pastas (Nível 2)

Podemos organizar ainda mais, criando pastas dentro de outras. Faremos isso para sites de bancos de imagens gratuitas.

4.1 - Criar a sub-pasta "Free PIcs"

  1. Pesquise e acesse o site Unsplash.
  2. Clique na estrela (☆) para favoritar.
  3. Na janela de favoritos, clique no menu "Pasta" e selecione "Outra pasta...".
  4. Na janela do editor de favoritos, certifique-se de que a pasta "DESENVOLVIMENTO WEB" esteja selecionada (azul).
  5. Clique no botão "Nova pasta".
  6. Digite o nome "Free PIcs" e clique em "Salvar". O site será salvo dentro da nova sub-pasta.

4.2 - Adicionar mais sites à sub-pasta

Agora, adicione os seguintes sites à sub-pasta que você acabou de criar:

  • Pexels
  • Freepik
  • Rawpixel
  • Pixabay
Como fazer: Para cada site, clique na estrela (☆). No menu "Pasta", navegue e selecione a sub-pasta "Free PIcs", que está dentro de "DESENVOLVIMENTO WEB". Clique em "Concluído".
